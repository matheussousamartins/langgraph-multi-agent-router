# LangGraph Multi-Agent Router

Sistema multi-agente construído com LangGraph onde um agente roteador
classifica a intenção do usuário e decide qual subgrafo especializado
deve ser executado.

## Arquitetura

Usuário → Router Graph → Subgrafo Especializado

Este projeto é compatível com LangGraph Studio para visualização
e depuração do fluxo de execução.
